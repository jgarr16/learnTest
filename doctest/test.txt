This is a sample doctest that checks Python's math operations. 

>>> 2 + 2

>>> 3 * 3

Now we're going to take some more of doctest's syntax for a spin.

>>> import sys
>>> def test_write():
...     sys.stdout.write("Hello\n")
...     return True
>>> test_write()
